# 卒業研究

## スタックメモリの可視化


### Quickstart
```bash
$ git clone https://github.com/s06181110/memoryVisualization
$ cd memoryVisualization
# XcodeのPythonを使うため xcrun で環境構築
$ xcrun python3 -m venv venv
# activate virtual environment
# When you want to end "deactivate"
$ . venv/bin/activate
# get some library
$ make prepare
# launch
$ make test
```
